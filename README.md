

AutoCRM Pro — Лэндинг для увеличения продаж автосалонов

Автоматизированный лендинг для презентации продукта AutoCRM Pro — AI-системы управления клиентским опытом для автомобильного бизнеса. Цель — повысить конверсии, привлечь клиентов и показать преимущества решения.



Описание проекта

Это одностраничный веб-сайт, выполненный на HTML и CSS с минимальным использованием JavaScript для интерактивных элементов и аналитики. 

- Главный заголовок с поддержкой подсказок (tooltip) для пояснений
- Продающий блок с описанием продукта и его ключевых преимуществ
- Секция с основными функциями и визуальными карточками
- Форма для получения бесплатного демо-доступа с валидацией
- Раздел гарантий и FAQ
- Анимация появления элементов при скролле
- Сбор базовой аналитики по кликам и времени на сайте

Структура файлов

- `index.html` — основной HTML файл со всей страницей и встроенными стилями и скриптами.
- Изображения в примере подгружаются через Placeholder (https://via.placeholder.com).

Запуск проекта

1. Сохраните файл как `index.html`.
2. Откройте файл в любом современном браузере (Chrome, Firefox, Edge, Safari).
3. Для локальной разработки не требуется сервер, страницы работает напрямую.

Ключевые особенности

- Адаптивный дизайн: используемые CSS-сетки и медиа-запросы обеспечивают корректное отображение на мобильных и десктопах.
- Tooltip-подсказки: поясняют важные элементы страницы для повышения доверия пользователя.
- Форма обратной связи: реализована проверка ввода и всплывающие уведомления.
- Аналитика: собираются данные о времени на сайте, кликах на основные кнопки, прокрутке страницы.
- Плавный скролл: при клике на ссылки-якоря страница плавно прокручивается к нужному блоку.


Настройка и расширение

- Чтобы заменить демонстрационное изображение, измените ссылку в теге `<img class="hero-image">`.
- Для подключения настоящей аналитики замените заглушки функций `trackEvent` и `trackConversion` на интеграцию с вашей системой.
- Добавьте реальную обработку формы вместо alert, используя серверный скрипт или сервис форм.
- Расширьте FAQ или другие разделы, добавив новые блоки по аналогии с существующими.


 Лицензия

Проект предназначен для обучения и демонстрации. Свободен для использования и модификации.
